# 단일 핵종 방사성 붕괴 시각화

이 저장소는 Streamlit 기반의 웹 애플리케이션으로, 단일 핵종의 방사성 붕괴를 시각화합니다.

## 실행 방법

1. 의존성 설치:
   ```bash
   pip install -r requirements.txt
   ```
2. Streamlit 앱 실행:
   ```bash
   streamlit run app/streamlit_app.py
   ```

## 이론적 배경

방사성 붕괴는 다음과 같은 지수 감쇠로 모델링됩니다.

- 붕괴 상수: \( \lambda = \frac{\ln 2}{T_{1/2}} \)
- 시간에 따른 핵자수: \( N(t) = N_0 e^{-\lambda t} \)

여기서 \(T_{1/2}\)는 반감기(초), \(N_0\)는 초기 핵자수(개)입니다.

## 주의사항 및 한계

- 수치 계산은 부동소수점 연산을 사용하므로 매우 긴 시간 범위나 극단적인 파라미터에서는 오차가 누적될 수 있습니다.
- 모델은 단일 핵종의 이상적인 붕괴만 다루며, 연쇄 붕괴나 외부 요인을 고려하지 않습니다.
